# HAR DESIGN

Une plateforme e-commerce moderne et élégante dédiée à la mode, développée avec React, TypeScript et Tailwind CSS. Ce projet inclut une boutique en ligne complète pour les clients et une interface d'administration pour la gestion des produits et des ventes.

## 🌟 Fonctionnalités

### 🛍️ Partie Client (Front-Office)
- **Catalogue Produits** : Navigation fluide à travers les collections (Vêtements, Chaussures, Parfums).
- **Panier d'achat** : Gestion dynamique du panier avec calcul automatique des totaux.
- **Expérience Utilisateur** : Animations soignées (Framer Motion) et design responsive.
- **Pages Informatifs** : 
  - *Accueil* : Mise en avant des produits phares.
  - *À Propos* : Présentation de la marque et des valeurs.
  - *Contact* : Formulaire de contact et informations de localisation.

### 👨‍💼 Partie Administration (Back-Office)
- **Tableau de Bord** : Vue d'ensemble des performances.
- **Gestion des Produits** : Ajout, modification et suppression des articles du catalogue.
- **Système de Caisse (POS)** : 
  - Interface dédiée pour les ventes en boutique physique.
  - Gestion des transactions (Ventes, Entrées, Dépenses).
  - Impression de factures thermiques professionnelles.
  - Suivi journalier des ventes.

## 🛠 Technologies Utilisées

- **Core** : React 18, TypeScript, Vite
- **Styles** : Tailwind CSS
- **Animations** : Framer Motion
- **Icônes** : React Icons, Lucide React
- **Graphiques** : ApexCharts (pour les statistiques)
- **PDF** : React-pdf (pour la génération de documents)
